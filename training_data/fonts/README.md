# Fonts

Place licensed CATIA/CTIS `.ttf` or `.otf` files here before generating target-specific OCR training data.

The generator will use fonts in this directory automatically:

```powershell
python tools\generate_digit_dataset.py
```

The official Dassault `CATIA DSISO1 font` download page redirects to 3DEXPERIENCE login, so the font file is not stored in this repository unless it is supplied from a licensed account.
